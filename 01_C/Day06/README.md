# Day 06 - Function

## 🎯 Goal
Learn how to create and use functions.

## 📚 Learned
- Function
- Parameter
- Function call
- Code reuse

## 💻 Example

```c
void printRPM(int rpm)
{
    printf("Engine RPM : %d\n", rpm);
}
```

## 🚗 Automotive Example

Separate RPM display into a function.

```c
printRPM(3000);
printRPM(4500);
printRPM(6000);
```

## ✍️ Memo

Functions make code cleaner and easier to reuse.
