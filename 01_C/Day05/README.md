# Day 05 - For Loop

## 🎯 Goal
Learn how to use the for loop in C.

## 📚 Learned
- for loop
- Initialization
- Condition
- Increment (++)
- Loop structure

## 💻 Example

```c
for(int i = 0; i < 5; i++)
{
    printf("%d\n", i);
}
```

## 🚗 Automotive Example

Use a for loop to simulate increasing engine RPM.

```c
for(int rpm = 1000; rpm <= 6000; rpm += 500)
{
    printf("%d\n", rpm);
}
```
