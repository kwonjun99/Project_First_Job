# const

const는 값을 변경하지 못하도록 만드는 키워드이다.

예시

```c
const int MAX_SPEED = 120;
```

이후

```c
MAX_SPEED = 200;
```

처럼 변경하려 하면 컴파일 오류가 발생한다.

자동차에서는

- 최대 속도
- 센서 개수
- 차량 길이

등 변하지 않는 값을 저장할 때 사용한다.

---

# 파일 쓰기

파일을 생성하고 데이터를 저장한다.

```c
FILE *fp;

fp = fopen("data.txt","w");

fprintf(fp,"Automotive Control");

fclose(fp);
```

실행하면

```
data.txt
```

파일이 생성된다.

---

# 파일 읽기

```c
char text[100];

fp = fopen("data.txt","r");

fscanf(fp,"%99[^\n]", text);

printf("%s\n", text);
```

저장된 문자열을 다시 읽어 화면에 출력한다.

---

# fopen()

```
"w"
```

쓰기 모드

파일이 없으면 생성하고,
있으면 내용을 모두 지운다.

```
"r"
```

읽기 모드

파일이 반드시 존재해야 한다.

---

# fclose()

열었던 파일을 닫는다.

파일 작업이 끝나면 반드시 호출해야 한다.
